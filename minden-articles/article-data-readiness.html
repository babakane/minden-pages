Data Readiness Assessment Framework

FREE 8 min read Business Intelligence January 2026

Before implementing any data analytics strategy, organizations must understand their current data infrastructure maturity. This framework provides a systematic approach to evaluate data readiness across six critical dimensions, complete with a Python-based automated scoring system.

The Six Dimensions of Data Readiness

Drawing from multi-sector analysis across telecom, energy, healthcare, and agriculture, we've identified six core components that determine an organization's ability to leverage data effectively:

  • Data Collection — Ability to gather data from multiple sources
  • Data Storage — Capacity and organization of data repositories
  • Data Processing — Tools and capabilities for data transformation
  • Data Security — Protection mechanisms and compliance
  • Data Quality — Accuracy, completeness, and reliability
  • Data Affordability — Cost-effectiveness of data operations

Implementation CODE SNIPPET

Python • data_readiness_assessment.py
import numpy as np
import matplotlib.pyplot as plt
from typing import Dict, List, Tuple

class DataReadinessFramework:
    """
    Six-dimensional data readiness assessment framework.
    Scores each component 0-100 for comprehensive evaluation.
    """
    
    DIMENSIONS = [
        'Data Collection',
        'Data Storage', 
        'Data Processing',
        'Data Security',
        'Data Quality',
        'Data Affordability'
    ]
    
    def __init__(self, scores: Dict[str, int]):
        """
        Initialize with dimension scores (0-100).
        
        Example:
            scores = {
                'Data Collection': 65,
                'Data Storage': 55,
                'Data Processing': 40,
                'Data Security': 70,
                'Data Quality': 45,
                'Data Affordability': 30
            }
        """
        self.scores = scores
        self.bottleneck_threshold = 50
        
    def calculate_overall_score(self) -> float:
        """Calculate weighted overall readiness score."""
        weights = [0.15, 0.15, 0.20, 0.20, 0.20, 0.10]
        weighted_sum = sum(
            self.scores[dim] * weight 
            for dim, weight in zip(self.DIMENSIONS, weights)
        )
        return round(weighted_sum, 2)
    
    def identify_bottlenecks(self) -> List[str]:
        """Identify dimensions scoring below threshold."""
        return [
            dim for dim in self.DIMENSIONS 
            if self.scores[dim] < self.bottleneck_threshold
        ]
    
    def generate_radar_chart(self) -> None:
        """Generate visualization of data readiness profile."""
        angles = np.linspace(0, 2 * np.pi, len(self.DIMENSIONS), endpoint=False)
        values = [self.scores[dim] for dim in self.DIMENSIONS]
        values += values[:1]  # Complete the circle
        angles = np.concatenate((angles, [angles[0]]))
        
        fig, ax = plt.subplots(figsize=(10, 8), subplot_kw=dict(projection='polar'))
        ax.plot(angles, values, 'o-', linewidth=2, color='#e94560')
        ax.fill(angles, values, alpha=0.25, color='#e94560')
        ax.set_xticks(angles[:-1])
        ax.set_xticklabels(self.DIMENSIONS, size=10)
        ax.set_ylim(0, 100)
        ax.set_title('Data Readiness Assessment', size=14, pad=20)
        ax.grid(True)
        plt.tight_layout()
        plt.show()
    
    def generate_report(self) -> Dict:
        """Generate comprehensive assessment report."""
        overall = self.calculate_overall_score()
        bottlenecks = self.identify_bottlenecks()
        
        report = {
            'overall_score': overall,
            'readiness_level': self._classify_readiness(overall),
            'dimension_scores': self.scores,
            'bottlenecks': bottlenecks,
            'recommendations': self._generate_recommendations(bottlenecks)
        }
        return report
    
    def _classify_readiness(self, score: float) -> str:
        """Classify overall readiness level."""
        if score >= 80: return 'Excellent'
        elif score >= 60: return 'Good'
        elif score >= 40: return 'Developing'
        else: return 'Critical'
    
    def _generate_recommendations(self, bottlenecks: List[str]) -> List[str]:
        """Generate improvement recommendations."""
        recommendations = {
            'Data Processing': 'Invest in ETL tools and data engineering capabilities',
            'Data Quality': 'Implement data validation and cleansing pipelines',
            'Data Affordability': 'Evaluate cloud storage options and optimize costs',
            'Data Collection': 'Expand sensor networks and data ingestion pipelines',
            'Data Storage': 'Modernize data warehouse architecture',
            'Data Security': 'Implement encryption and access control frameworks'
        }
        return [recommendations[b] for b in bottlenecks if b in recommendations]


# Example Usage
if __name__ == "__main__":
    # Sample assessment for an emerging market telecom
    assessment_scores = {
        'Data Collection': 65,
        'Data Storage': 55,
        'Data Processing': 40,
        'Data Security': 70,
        'Data Quality': 45,
        'Data Affordability': 30
    }
    
    framework = DataReadinessFramework(assessment_scores)
    report = framework.generate_report()
    
    print(f"Overall Data Readiness Score: {report['overall_score']}/100")
    print(f"Readiness Level: {report['readiness_level']}")
    print(f"\nBottlenecks Identified: {len(report['bottlenecks'])}")
    for bottleneck in report['bottlenecks']:
        print(f"  - {bottleneck}: {assessment_scores[bottleneck]}%")
    
    print("\nRecommendations:")
    for rec in report['recommendations']:
        print(f"  → {rec}")
    
    # Generate visualization
    framework.generate_radar_chart()

Key Insights from the Framework

The radar chart visualization immediately highlights areas of strength and weakness. In our example above, Data Security (70%) and Data Collection (65%) are relative strengths, while Data Affordability (30%) and Data Processing (40%) require urgent attention.

💡 Pro Tip: Weighting Strategy

The default weights prioritize Data Processing, Security, and Quality (20% each). Adjust these based on your sector:

• Healthcare: Increase Security weight to 30%
• IoT/AgriTech: Increase Collection weight to 25%
• Startups: Increase Affordability weight to 20%

Applying to Your Environment

This framework is designed to be location and sector-agnostic. Whether you're assessing a rural healthcare network, an urban telecom infrastructure, or an agricultural IoT deployment, the six dimensions remain consistent:

  1. Score each dimension based on your current capabilities (0-100)
  2. Identify bottlenecks — dimensions below 50% need immediate investment
  3. Generate recommendations using the built-in suggestion engine
  4. Track progress quarterly by re-running the assessment
